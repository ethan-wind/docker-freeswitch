
WebSocketpp只是一个库，本身不需要搭建什么环境，只要新建的项目引入相关的库就行。但是WebSocketpp依赖于boost库

1、安装boost库
cd boost_1_63_0
./bootstrap.sh
./b2
./b2 install

提示 error while loading shared libraries: libboost_thread.so.1.63.0: cannot open shared object file: No such file or directory
解决方法  sudo ldconfig /usr/local/lib


2.centos curl安装
    1、解压
         下载到的压缩包为curl-7.51.0.tar.gz，使用命令  tar -zxvf curl-7.51.0.tar.gz   解压。
     2、进入解压出的目录curl-7.51.0，执行
       （1）sudo ./configure --with-openssl
       （2）sudo make  
       （3）sudo make install

3. cmake 安装
./configure
make
sudo make install
cmake --version

4.配置websocketpp
build.sh文件里需要改为websocketpp的存放目录。
